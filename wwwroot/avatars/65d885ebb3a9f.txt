# Course_Azure
# (технічне завдання)
# 
#	Сайт, для розміщення приватних оголошень з продажу будь-яких речей в тому числі уживаних
#
# Головна сторінка повинна відображати:
#	1. Кнопка входу/реєстрації
#	2. Кнока подачі оголошення
#	3. Катергорії товарів, що продаються
#	4. Блок для рекламної вставки
#
# База даних має відображати:
#	- id Продавця - Повинна бути таблиця продавців (id, Name, Telephone)
#	- id Товару - Повинна бути таблиця товарів (id, id_salesman, Category, Name, Description, Photo, Сost, Стан[новий/уживаний])
#	- id Замовлення - Повинна бути таблиця замовлень (id, id_product, Name, Telephone, Delivery address, Статус[виконано/не виконано])
# 
# -> Кнопка входу:
#	- знаходиться у верхньому полі
#	- відкривається у вигляді модального вікна
#	- має містити:
#		* поле логін
#		* поле пароль
#		* кнопка "вхід"
#		* посилання у вигляді кнопки на форму реєстрації (окрема сторінка)
# 
# -> Кнока подачі оголошення:
#	- знаходиться у верхньому полі (але краще щоб зявлялась після входу)
#	- відкривається у вигляді модального вікна
#	- містить форму додавання товару (маєтья на увазі додавання товару до бази даних)
#	- кнопки:
#		* "Додати" - дадає товар до бази даних
#		* "Відміна" - очищає форму
#		* "Вихід" - закриває модальне вікно без збереження даних форми
# 
# -> Катергорії товарів, що продаються:
#	- 1й рядок, 3 комірки:
#		* Безкоштовні
#		* Одяг
#		* Дитячий світ
#	- 2й рядок, 3 комірки:
#		* Дім і сад
#		* Автотовари
#		* Електроніка
#	- 3й рядок, 1 велика комірка:
#		* Всі товари
# 
# -> Блок для рекламної вставки:
#	- Картка із зображенням реклами
#	- Посилання на сайт рекламодавця
# 
# Після вибору категорії товару з бази даних завантажується товар з обраною категорєю.
# Товар на сторінці розміщується у вигляді карток з кнопкою (зображення товару, назва, ціна).
# При натисканні кнопки з'являється опис товару.
# 
# При виборі товару відкривається сторінка товару:
#	- зображення товару збільшене
#	- назва, опис, ціна
#	- інформація про продавця
#	- кнопка купити:
# 		* відкриває модальне вікно
#		* форма для оформлення доставки (Ім'я, телефон та адреса доставки покупця)
#		* кнопки:
#			- "Купити" - дадає товар до таблиці Замовлень бази даних
#			- "Відміна" - очищає форму та закриває модальне вікно
# 
# -> Після входу в особистий кабінет
#	- відображаються особисті дані
#	- доступ до таблиці товарів, що продаються авторизованим користувачем
#	- доступ до таблиці замовлень, куплених товарів авторизованого користувача
